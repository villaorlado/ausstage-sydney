Mechanism being tested

The pre-shock field is intentionally held constant. After the funding shock, more performers and productions enter. If these new productions mostly draw from familiar neighbourhoods, clustering can remain high. If the mixer parameter is high, new productions create bridges across previously separate neighbourhoods. Those bridges increase connectivity but do not immediately create closed triangles, so average clustering falls.
